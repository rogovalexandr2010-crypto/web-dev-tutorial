# WebDev Academy - Современная образовательная платформа

Интерактивная платформа для изучения HTML, CSS и JavaScript с нуля. Идеально подходит для абсолютных новичков, которые хотят освоить веб-разработку.

## 🎯 Основные возможности

### 📚 Структурированное обучение
- **HTML модуль** - 3 полноценных урока с практическими примерами
- **CSS модуль** - 3 урока о стилизации и дизайне
- **JavaScript модуль** - 3 урока о программировании
- Каждый урок содержит теорию, примеры кода и практические задания

### 💻 Встроенный редактор кода
- Живой редактор для HTML, CSS и JavaScript
- Мгновенный предпросмотр результата в real-time
- Автоматическое сохранение кода в браузер
- Поддержка трёх языков с переключением между ними

### ✅ Практические задания
- 15+ интерактивных задач для закрепления материала
- Постепенное повышение сложности
- Система подсказок и решений
- Проверка выполнения заданий

### 📊 Отслеживание прогресса
- Сохранение прогресса по каждому модулю
- Система достижений и мотивации
- Подсчёт завершённых уроков и задач
- Отслеживание серии дней обучения

### 🎨 Современный интерфейс
- Светлая и тёмная темы оформления
- Адаптивный дизайн для мобильных устройств
- Интуитивная навигация
- Красивые анимации и переходы

## 🚀 Быстрый старт

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/rogovalexandr2010-crypto/web-dev-tutorial.git
cd web-dev-tutorial

# Установите зависимости
npm install

# Запустите сервер
npm start
```

Сервер запустится на `http://localhost:3000`

### Требования
- Node.js 14+
- npm или yarn

## 💾 Сохранение данных

Платформа использует Browser LocalStorage для сохранения:
- Выбранной темы (светлая/тёмная)
- Написанного кода в редакторе
- Прогресса обучения
- Информации о пользователе
- Завершённых уроков и задач

## 🔐 Безопасность

- Код пользователя выполняется в изолированном iframe
- Нет отправки персональных данных на сервер
- Все данные хранятся локально в браузере
